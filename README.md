## О проекте

ЗаботаОнлайн — это веб-сервис, который помогает семьям организовать уход за пожилым родственником. Система объединяет пациента, родственников и лечащего врача в едином цифровом пространстве.

**Проблема:** уход за пожилыми людьми часто распределён между несколькими родственниками, которые живут отдельно. Важная информация (пропуск лекарств, повышение давления, жалобы на самочувствие) фиксируется несистемно — в бумажных дневниках или мессенджерах.

**Решение:** централизованный дневник самонаблюдения с уведомлениями и разграниченным доступом по ролям.

## Функциональность

### 👴 Пациент
- Ввод артериального давления и пульса с автоматической классификацией (норма / повышенное / критично)
- Расписание лекарств с отметками «Принял» / «Пропустил»
- Просмотр истории с фильтрами по датам и типу записей
- Код приглашения для подключения родственников и врача

### 👨‍👩‍👧 Родственник
- Подключение к пациенту по коду приглашения (поддержка нескольких пациентов)
- График давления за 7 / 30 дней с линией нормы
- График приверженности терапии по неделям
- Уведомления о критических событиях
- Отправка сообщений пациенту

### 🩺 Врач
- Дашборд с ключевыми показателями: среднее давление, приверженность, критические эпизоды
- График динамики давления с линией нормы пациента
- Список жалоб пациента
- Рекомендации трёх уровней (информация / рекомендация / срочно)
- Фиксация решения «рекомендован очный приём»

### ⚙️ Общее
- Страница настроек профиля (имя, телефон, пароль, норма давления)
- Полная история с фильтрами по датам и типу записей
- Журнал аудита всех действий (требование ФЗ-152)

## Быстрый старт

### Требования
- [XAMPP](https://www.apachefriends.org/) с PHP 8.2+

### Установка

1. Склонируй репозиторий в папку `htdocs`:
```bash
git clone https://github.com/Olivvvvvka/eldercare.git C:/xampp/htdocs/eldercare
```

2. Запусти Apache в XAMPP Control Panel

3. Инициализируй базу данных:
```
http://localhost/eldercare/setup.php
```

4. Открой приложение:
```
http://localhost/eldercare/frontend/index.html
```
